\section{Roles y permisos}

\subsection{Descripcion}

Este documento reservara la matriz funcional de permisos por rol del sistema.

\subsection{Alcance}

Aplica a usuarios participantes, admins y super admins.

\subsection{Definiciones}

\begin{itemize}
  \item \textbf{Permiso}: accion autorizada sobre una capacidad del sistema.
\end{itemize}

\subsection{Reglas}

\begin{itemize}
  \item El modelo de permisos seguira `Least Privilege`.
  \item Todo permiso administrativo sensible debe poder auditarse.
\end{itemize}

\subsection{Casos borde}

\begin{itemize}
  \item Pendiente de aterrizar la matriz accion por accion.
\end{itemize}

\subsection{Invariantes}

\begin{itemize}
  \item Ningun rol recibe permisos mas amplios de los necesarios para su operacion.
\end{itemize}

\subsection{Preguntas abiertas}

\begin{itemize}
  \item Confirmar diferencia exacta de permisos entre `admin` y `super admin`.
\end{itemize}
