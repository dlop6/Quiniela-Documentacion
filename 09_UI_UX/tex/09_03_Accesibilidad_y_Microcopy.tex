\section{Accesibilidad y microcopy}

\subsection{Descripcion}

Este documento reservara lineamientos de accesibilidad y tono funcional del producto.

\subsection{Alcance}

Aplica a textos, mensajes de estado, errores y accesibilidad basica.

\subsection{Definiciones}

\begin{itemize}
  \item \textbf{Microcopy}: texto corto orientado a guiar acciones, estados o errores.
\end{itemize}

\subsection{Reglas}

\begin{itemize}
  \item El lenguaje debe ser simple y no tecnico.
  \item La accesibilidad no debe depender solo de color.
\end{itemize}

\subsection{Casos borde}

\begin{itemize}
  \item Pendiente de aterrizar mensajes por flujo y por error.
\end{itemize}

\subsection{Invariantes}

\begin{itemize}
  \item Los mensajes no deben ocultar reglas reales del sistema.
\end{itemize}

\subsection{Preguntas abiertas}

\begin{itemize}
  \item Confirmar estilo final de mensajes y tono institucional.
\end{itemize}
