\section{Email y notificaciones}

\subsection{Descripcion}

Este documento reservara los eventos del sistema que deben producir correos o notificaciones operativas.

\subsection{Alcance}

Aplica a comunicaciones minimas previstas para v1.

\subsection{Definiciones}

\begin{itemize}
  \item \textbf{Notificacion accionable}: mensaje que impulsa una accion concreta del usuario o del admin.
\end{itemize}

\subsection{Reglas}

\begin{itemize}
  \item Las notificaciones de v1 deben ser pocas, claras y funcionales.
  \item La mensajeria debe respetar estados reales del sistema.
\end{itemize}

\subsection{Casos borde}

\begin{itemize}
  \item Pendiente de aterrizar los triggers definitivos y el copy funcional.
\end{itemize}

\subsection{Invariantes}

\begin{itemize}
  \item Ninguna notificacion debe prometer un estado que no haya sido persistido y auditado cuando aplique.
\end{itemize}

\subsection{Preguntas abiertas}

\begin{itemize}
  \item Confirmar lista final de emails obligatorios y opcionales para v1.
\end{itemize}
