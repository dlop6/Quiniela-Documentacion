\section{API de resultados}

\subsection{Descripcion}

Este documento reservara la integracion conceptual con la API externa de resultados.

\subsection{Alcance}

Aplica a consumo funcional de resultados y mapeo hacia el dominio interno.

\subsection{Definiciones}

\begin{itemize}
  \item \textbf{Proveedor de resultados}: servicio externo que entrega datos oficiales o candidatos a oficiales.
\end{itemize}

\subsection{Reglas}

\begin{itemize}
  \item La API externa se adapta al dominio; el dominio no adopta directamente el modelo externo.
  \item Todo resultado externo debe poder rastrearse hasta su fuente.
\end{itemize}

\subsection{Casos borde}

\begin{itemize}
  \item Pendiente de completar al seleccionar proveedor definitivo y su contrato.
\end{itemize}

\subsection{Invariantes}

\begin{itemize}
  \item El proveedor externo no sustituye la autoridad documental de las reglas del juego.
\end{itemize}

\subsection{Preguntas abiertas}

\begin{itemize}
  \item Confirmar proveedor oficial, frecuencia operativa y restricciones de uso.
\end{itemize}
