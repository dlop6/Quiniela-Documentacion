\section{Principios UI UX}

\subsection{Descripcion}

Este documento reservara los principios funcionales de interfaz y experiencia para la v1.

\subsection{Alcance}

Aplica a usuario participante y administracion.

\subsection{Definiciones}

\begin{itemize}
  \item \textbf{Accion principal}: CTA dominante de una pantalla.
\end{itemize}

\subsection{Reglas}

\begin{itemize}
  \item Cada pantalla debe tener una accion principal clara.
  \item La UX debe favorecer claridad, accesibilidad y baja friccion.
\end{itemize}

\subsection{Casos borde}

\begin{itemize}
  \item Pendiente de expandir con estados y patrones UI finales.
\end{itemize}

\subsection{Invariantes}

\begin{itemize}
  \item Ninguna decision visual puede contradecir una restriccion funcional del dominio.
\end{itemize}

\subsection{Preguntas abiertas}

\begin{itemize}
  \item Confirmar patrones definitivos de navegacion y wizard.
\end{itemize}
